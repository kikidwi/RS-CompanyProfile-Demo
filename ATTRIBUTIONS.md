# Atribusi Proyek

Website Company Profile RS Utama Demo ini tidak terlepas dari peran berbagai pustaka (libraries), aset, dan perangkat lunak open-source.

## Komponen & Framework
*   **[React](https://react.dev/)** – Library utama untuk membangun antarmuka pengguna.
*   **[Vite](https://vitejs.dev/)** – Build tool frontend yang super cepat.
*   **[Tailwind CSS](https://tailwindcss.com/)** – Framework utility-first CSS untuk proses styling yang responsif dan efisien.
*   **[React Router](https://reactrouter.com/)** – Library standar untuk navigasi dan routing di React.
*   **[Lucide React](https://lucide.dev/)** – Koleksi ikon SVG yang indah dan konsisten (digunakan di bawah lisensi ISC).

## Aset Visual & Media
*   **[Unsplash](https://unsplash.com/)** – Semua gambar gedung, ilustrasi fasilitas, dan banner layanan bersumber dari Unsplash (digunakan di bawah [Lisensi Unsplash](https://unsplash.com/license)).
*   **[RandomUser.me API](https://randomuser.me/)** – Menyediakan placeholder wajah untuk foto profil direktori dokter (digunakan hanya sebagai purwarupa/demo).
*   **[Google Fonts](https://fonts.google.com/)** – Penyedia font **Karla** yang digunakan sebagai tipografi utama (lisensi SIL Open Font License).

## Desain Awal (Referensi)
*   Sebagian tata letak terinspirasi oleh figma open source atau desain publik rumah sakit untuk keperluan edukasi dan portofolio.
